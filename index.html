사업자 휴/폐업조회

* 사용방법
  • 1. API키 값을 입력합니다.(복사&붙여넣기)
  • 2. 여러 건을 변환하는 경우 (2-1. 파일로 변환하기)
    1. 업로드용 엑셀 파일을 다운받습니다.
      여기를 눌러 다운로드받기
    2. 업로드파일 형식에 맞춰, 변환하고자 하는 값을 입력합니다.
      1. 가맹점관리번호 : 임의의 유니크한 숫자값 기입
      2. 사업자관리번호 : 10자리 숫자로, 숫자만 기입
    3. [파일 업로드] 버튼을 눌러, 파일을 업로드 합니다.
    4. 업로드가 완료되면, [변환하기] 버튼을 통해 변환을 시작합니다.
    5. 변환이 완료된 후, [다운로드] 버튼을 통해 결과 파일을 다운받습니다.
  • 3. 한 건씩 변환하는 경우 (2-2. 1건씩 변환하기)
    1. (-)을 제외한 10자리 숫자값만을 기입하여, [변환하기]버튼을 클릭합니다.
